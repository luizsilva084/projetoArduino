# Projetos com Arduino

Bem-vindo ao meu repositório de projetos com Arduino 🚀

Aqui compartilho experiências, testes e projetos desenvolvidos durante meus estudos em eletrônica e programação embarcada utilizando Arduino.

## 📌 Projetos disponíveis

- Circuitos básicos de eletrônica
- Experimentos com sensores e componentes

- Bateria 9V
